Система бронирования отелей

Консольное приложение для управления бронированием номеров в отелях с возможностью поиска, статистики и визуализации данных.



Описание

Данное консольное приложение моделирует работу системы бронирования номеров в отелях. Программа работает с табличными данными на базе pandas, ведёт лог действий пользователя и позволяет сохранять данные в CSV.

Поддерживаемые отели:
- Marriott
- Hilton
- Hyatt
- Radisson
- Azimut

Типы номеров:
- Стандарт
- Люкс
- Полулюкс
- Семейный

Статусы брони:
- Забронировано
- Заселён
- Выселен



Функции

Работа с данными

- Генерация начальных данных — 10 случайных броней при запуске
- Просмотр всех броней — удобная таблица со всей информацией
- Добавление брони — с проверкой доступности номера на выбранные даты
- Удаление брони — по уникальному ID
- Экспорт в CSV — сохранение текущего состояния в result.csv

Поиск и фильтрация
- По фамилии гостя
- По названию отеля
- По типу номера
- По статусу брони

Управление статусами
- Изменение статуса брони (Забронировано → Заселён → Выселен)
- Проверка конфликтов при добавлении новых броней

Статистика и аналитика
- Средняя цена за ночь по каждому типу номера
- Общая выручка по отелям
- Топ-3 самых дорогих брони
- Средняя продолжительность пребывания
- Поиск броней длиннее указанного количества ночей

Визуализация
- Диаграмма выручки по отелям
- Диаграмма распределения броней по типам номеров
- График средней цены за ночь по отелям

Логирование
Все ключевые действия пользователя записываются в файл log.txt с временными метками:
- Запуск и завершение программы
- Добавление/удаление броней
- Изменение статусов
- Ошибки ввода
- Поисковые запросы
- Создание графиков



Используемые технологии

- Python 3.8+
- pandas — работа с табличными данными
- numpy — генерация случайных данных
- matplotlib — построение графиков и диаграмм
- datetime — работа с датой/временем для логов
- Стандартные модули: random
